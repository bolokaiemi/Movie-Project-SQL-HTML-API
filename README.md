# Movie Library Manager
Movie Library Manager is a web application that lets users manage their personal movie collections. It pulls data from the OMDB API so people can search for films, view details, and create a portfolio website for their collection.
## Features
Search for movies using the OMDB API to get real-time details, view movie information like title, release year, genre, plot, and ratings, and manage your collection by adding, reading, updating, and deleting movies
